About Me

안녕하세요. 백엔드 개발자 박진주입니다.

저는 주어진 환경에 의존하지 않고 주도적으로 문제를 해결하며 성장하는 개발자입니다.

이전 직장에서 사수가 없는 교대근무 속에서도 출퇴근 시간을 조정하며 불량 시료를 활용한 테스트와 관련자료를 찾아보며 업무를 익혔습니다. 모르는 부분을 정리하고 적극적으로 질문하며 문제 해결 능력을 키웠고, 그 결과 핵심 인력으로 자리 잡아 업체가 아니면 해결이 어렵다고 판단된 문제들까지 스스로 해결할 수 있는 역량을 갖추게 되었습니다.

데이터 분석과 오류 해결을 하는 과정에서 코드에 매력을 느꼈으며, 이후 본격적으로 Java 및 Spring 기반 백엔드 개발을 학습하며 직접 코드를 이해하고 문제를 해결하는 과정에서 개발자로서의 길을 확신하게 되었습니다.

현재 Java, Spring Boot, Spring MVC, JPA를 활용한 백엔드 개발을 익히고 있으며, Oracle DB를 기반으로 한 데이터 모델링, 쿼리 최적화, API 설계 및 RESTful 서비스 구축 경험을 쌓고 있습니다. 또한 프론트엔드와의 원활한 협업을 위해 HTML, CSS, JavaScript 등의 기술도 학습 중입니다.

저는 "쿼카의 사람 버전"이라 불릴 만큼 붙임성과 친화력이 뛰어난 개발자입니다.

긍정적인 관계를 형성하는 것을 중요하게 생각하며, 전 직장에서 부서뿐만 아니라 외부 파견 인력, 신입사원 등 다양한 사람들과 긍정적인 관계를 맺었습니다. 덕분에 10년이 지난 지금도 인연을 이어가고 있으며 저는 좋은 관계와 분위기 속에서 최고의 결과물이 나온다고 믿고 팀워크를 중요하게 생각합니다.

앞으로도 꾸준히 새로운 기술에 도전하고 문제 해결 능력을 키우며 협업을 통해 성장하는 개발자가 되겠습니다.

매일 어제보다 나은 개발자가 되기 위해 노력하며, 궁극적으로는 실질적인 가치를 제공하고 믿고 맡길 수 있는 풀스택 개발자로 인정받는 것이 목표입니다.
